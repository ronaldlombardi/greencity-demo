# 🌿 GreenCity AI Agent - Demo Interactiva

Diagnóstico inteligente de espacios verdes urbanos usando imágenes satelitales e inteligencia artificial.

## 🔗 Demo en vivo

Abrí la demo sin instalar nada: [GreenCity Demo](https://ronaldlombardi.github.io/greencity-demo/)

## 📍 Ciudades analizadas

- **Villa María - Villa Nueva** (Córdoba, Argentina) - 120.000 hab
- **San Francisco** (Córdoba, Argentina) - 62.000 hab

## 📊 Indicadores evaluados

- Cobertura del suelo (ESA World Cover 2020)
- Accesibilidad a espacios verdes (distancia <300m - OMS)
- Cobertura arbórea urbana
- Comparativa regional entre ciudades
- Recomendaciones basadas en datos

## 🛠️ Tecnología

- Google Earth Engine
- ESA World Cover 2020
- Leaflet.js + Chart.js
- GitHub Pages

git clone https://github.com/ronaldlombardi/greencity-demo.git
